Documentation

Finally; all the Popcorn time data you'll ever need, in once place, and easily accessible through a modern RESTful API.

Shows

URLDescription
http://cloud-services-assignment.herokuapp.com/showsGet all shows

Show

URLDescription
http://cloud-services-assignment.herokuapp.com/shows/{imdb_id}Get specific show by IMDB ID

Properties

PropertyDataTypeDescription
_idStringUnique Identifier for the show
imdb_idStringIMDB ID, same ID as used on the IMDB website
tvdb_idStringTVDB ID, same ID as used on the TVDB website
titleStringThe show's title
yearStringThe release year
slugStringThe show's slug
synopsisStringA short description of the most important information
runtimeStringThe show's runtime
rating*ObjectContains properties showing the show's rating
countryStringThe show's origin country
networkStringWhere the show can be watched
air_dayStringWhich day the show airs
air_timeStringWhich time the show airs
statusStringThe show's status, E.G.: ended or returning series
num_seasonsIntegerThe number of seasons released
last_updatedNumberLast updated timestamp
images*ObjectContains properties with URL's showing images related to the show

Rating

PropertyDataTypeDescription
hatedIntegerNumber of times the show was marked 'hated'
lovedIntegerNumber of times the show was marked 'loved'
votesIntegerThe total amount of votes for this show
watchingIntegerThe amount of people currently watching (amount of peers)
percentageIntegerCurrent score E.G.: 84% = 8.4
bannerStringUrl with a banner

Images

PropertyDataTypeDescription
fanartStringUrl with fan art
posterStringUrl with a poster
bannerStringUrl with a banner

Episodes

URLDescription
http://cloud-services-assignment.herokuapp.com/shows/{imdb_id}/episodesGet all episodes belonging to a show

Episode

URLDescription
http://cloud-services-assignment.herokuapp.com/shows/{imdb_id}/episode?season={season}&episode={episode}Get episodes belonging to show

Query parameters

PropertyDataTypeDescription
seasonIntegerTo what season this episode belongs
episodeIntegerThe episode number

Will return a single episode when both season & episode quary parameters are set.

Properties

PropertyDataTypeDescription
tvdb_idStringTVDB ID, same ID as used on the TVDB website
seasonIntegerTo what season this episode belongs
episodeIntegerThe episode number
titleStringThe episode title
overviewStringShort summary about the episode
date_basedBooleanEpisodes for this season are released all at once or with a given period in between
first_airedIntegerTimestamp of when the episode first aired
watchedObjectContaining a single boolean named "watched" with wether the user has watched the episode
torrents*Object arrayContaining a list of torrents and urls to them

Torrents

PropertyDataTypeDescription
urlStringurl to torrent file / magnet link
seedsIntegerNumber of current seeders
peersIntegerNumber of current peers
providerStringUploader name
